# Sanchez_word_cloud
☁️ СИБУР Форум — Интерактивное облако слов

СИБУР Word Cloud — это легковесное, полностью автономное одностраничное приложение (SPA) для генерации интерактивных облаков слов на основе ответов участников форумов, опросов или мозговых штурмов.

Проект разработан с учетом строгих корпоративных гайдлайнов (Брендбук СИБУР, релиз 2.0).

✨ Ключевые особенности

Zero-Dependency: Не требует установки пакетов (npm), Node.js или базы данных. Вся логика, парсинг и рендеринг выполняются на стороне клиента (в браузере).

Продвинутый парсинг текста: Автоматическая очистка ввода от цифр, пунктуации и спецсимволов. Разделение по запятым, пробелам и переносам строк.

Встроенный фильтр стоп-слов: Автоматическое исключение предлогов, союзов и местоимений (более 50+ слов в словаре).

Алгоритм Спирали Архимеда: Использование Canvas API (measureText) для точного расчета bounding-box'ов слов и плотной укладки без наложений (pixel-perfect collision detection).

Brand-Compliant: Строгое соответствие корпоративному стилю. Использование официальной палитры (ДНК СИБУР, Мятный, Ярко-оранжевый), шрифта Roboto и модульной сетки без использования запрещенных эффектов свечения.

🚀 Быстрый старт

Проект не требует сборки. Для запуска достаточно открыть файл в любом современном браузере.

Склонируйте репозиторий:

git clone [https://github.com/your-username/sibur-word-cloud.git](https://github.com/your-username/sibur-word-cloud.git)


Откройте файл sibur_word_cloud.html в браузере (Chrome, Safari, Firefox, Edge).

Вставьте текст в поле ввода и нажмите «Показать облако».

🧠 Как это работает (Под капотом)

1. Обработка и нормализация текста

Скрипт перехватывает сырой текст и пропускает его через цепочку преобразований:

Разделение с помощью RegEx: /[\n,;\s]+/

Удаление небуквенных символов: /[^а-яёa-z\-]/gi

Очистка от "висячих" дефисов: /^-+|-+$/g

Фильтрация по словарю стоп-слов (структура Set для O(1) поиска).

2. Рендеринг (Позиционирование слов)

Для каждого слова вычисляется его частота появления. Размер шрифта масштабируется нелинейно на основе максимального количества повторений.
Алгоритм ищет свободное место для слова, двигаясь из центра экрана по спирали Архимеда:

// Упрощенная концепция алгоритма:
let angle = Math.random() * Math.PI * 2;
let r = 0;
for (let i = 0; i < 2000; i++) {
  const x1 = cx + r * Math.cos(angle) - w / 2;
  const y1 = cy + r * Math.sin(angle) - h / 2;
  if (!intersects(x1, y1, x1 + w, y1 + h)) {
    // Место найдено, рендерим слово
    break;
  }
  r += step; angle += 0.25;
}


3. Интерактивность и UX

Приложение поддерживает автоматический перерасчет позиций при изменении размера окна (используется паттерн debounce). При наведении на слово появляется динамичный тултип с расчетом абсолютного и относительного количества упоминаний.

🎨 Кастомизация

Добавление новых стоп-слов

Если специфика вашего форума подразумевает частое использование мусорных слов (например, сленга), вы можете легко добавить их в словарь. Откройте .html файл и найдите константу STOP_WORDS:

const STOP_WORDS = new Set([
  'и', 'а', 'но', 'да', ... ,
  'ваше_новое_слово_1', 'ваше_новое_слово_2'
]);


Настройка палитры

Цветовая схема хранится в массиве COLORS и CSS-переменных :root. По умолчанию используются HEX-коды из брендбука СИБУР.
